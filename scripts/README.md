# MTG準備

## 「MTG準備」と送ったときの動き

**Slack・Comet（ブラウザ）の起動は行いません。**

代わりに、チャットで **「MTG準備」** と送ると、AIが次のことを行います。

- 前回の議事録（議事録/ フォルダの直近）・意思決定ログ・設定/次回MTG.md を読む
- **社長が準備できているかの振り返り**を出力する（社長の宿題・次回で確認することのチェックリスト、準備状況の整理）

Notta・Notion・Slack は必要に応じて手動で開いてください。

## open_mtg_prep.sh

アプリの起動は行わず、MTG準備の振り返りはチャットで行う旨を表示するだけのスクリプトです。ターミナルで実行しても、チャットで「MTG準備」と送ったときの動作には影響しません。
