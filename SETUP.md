# gijiroku — はじめての方へ

> ブレーメングループ 議事録・ナレッジ管理リポジトリ

---

## このリポジトリでできること

Cursorで開くと、AIがこのリポジトリの構造・命名規則・会議ルールを把握した状態で動きます。

```
「今日のMTG議事録を作って」
「先月の意思決定をまとめて」
「このナレッジをファイルに保存して」
```

---

## セットアップ確認（1分）

Cursorで開いたら、チャットに入力：

```
このリポジトリの構成を教えて
```

フォルダ構成と使い方を説明できれば準備完了です。

---

## 新しいPCでのセットアップ

```bash
# GitHubにログイン（オーナーのみ・初回）
gh auth login

# リポジトリを取得
git clone https://github.com/komatsu-matsuko/gijiroku.git
cursor gijiroku/
```

---

## 基本的な使い方

### 議事録を作る

```
「[日付]の[会議名]の議事録を作って」
→ AIがMarkdown形式で生成
→ 確認後「議事録フォルダに保存して」と言うと保存
```

### レポートを書く

```
「今週の週次レポートを作って」
→ 内容確認後「週次レポートフォルダに保存して」
```

### ナレッジを追加する

```
「[内容]をナレッジに追加して」
```
