# 意思決定ログ

社長が下した判断を「誰に・何を・いつまで」で記録し、伝達漏れ防止と実行の追いやすさを確保します。

## ログの追加方法

- チャットで **「意思決定を記録」** と送り、続けて「誰に・何を・いつまで」を話す。
- または議事録の決定事項を共有し、**「これを意思決定ログに追加」** と送る。

AIが `意思決定ログ/YYYY-MM.md` に追記します（月ごとファイル）。

## 1件のフォーマット

```markdown
### YYYY-MM-DD タイトル（簡潔に）

| 項目 | 内容 |
|------|------|
| 誰に | 担当者・部署 |
| 何を | 判断内容・指示 |
| いつまでに | 期限 |
| 補足 | あれば |
```
