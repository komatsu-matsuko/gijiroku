# 設定（後ほど修正用）

ここにある設定値を変えると、リマインダー送信先や次回MTG日付などが切り替わります。

| ファイル | 内容 | 修正したいとき |
|----------|------|----------------|
| **slack.md** | リマインダー送信先（C0ADM5X1XJA）・議事録・FB送信先（社長のSlack D01KDC0NN7M） | 本番用に変えるとき |
| **次回MTG.md** | 次回幹部MTGの予定日 | 毎回のMTG後に更新 |

---

## 変更手順

1. 該当ファイルを開いて値を編集・保存する。
2. 「リマインダー作成」「会議前ブリーフィング」などは、次回実行時から新しい設定を参照する。

「後ほど修正」と言ったときは、このフォルダのファイルを確認してください。

---

## 社員の増減・制度変更時の更新一覧

**どのファイルをいつ更新するか**は、議事録フォルダ直下の **`議事録_保守・更新ガイド.md`** にまとめてあります。社員が増えた・役職が変わった・評価制度を変えたときなどは、そちらを開いて該当箇所を更新してください。
