# 経営アドバイザー

幹部MTGの内容を **経営アドバイザー視点** で振り返り、役員・幹部の未熟な箇所や改善点を指摘するためのフォルダです。

## 役割

- 会議のやりとりから読み取れる **幹部の未熟な点**（判断の遅れ、共有不足、意思決定の曖昧さなど）を指摘する。
- 組織としての改善点（進め方・意思決定の質・コミュニケーション）を簡潔にまとめる。
- 社長の意思決定の材料として参照する。

## 出力の保存

「議事録を作成」実行時に、**経営アドバイザー所見** をこのフォルダに保存する想定。

- ファイル名例：`経営アドバイザー/YYYY-MM-DD_幹部MTG_所見.md`
- 内容：会議日・指摘ポイント（箇条書き）・改善の提案（短く）

## 注意

- 所見は社長向けのフィードバックであり、役員FBとは別物。役員FBは評価制度に基づく個別のフィードバック。
