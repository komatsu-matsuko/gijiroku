# ナレッジ

幹部MTGの議事録要約・決定事項・アクションを蓄積し、検索しやすくします。

## 保存形式

- 1会議あたり `YYYY-MM-DD_幹部MTG.md` のようなファイルで保存。
- 内容：日付・会議名・決定事項・アクション（誰が・何を・いつまでに）・参照用キーワード。

## Notion・Spreadsheet への取り込み

検索や一覧管理のために、NotionやGoogleスプレッドシートに取り込む場合：

- **「ナレッジをエクスポート」** とチャットで送る。
- 直近のナレッジファイルを参照し、次の形式で出力する：
  - **CSV**：列は `日付, 会議名, 決定事項, アクション, キーワード`。1行1会議または1行1アクション。
  - **表形式（Markdown）**：Notionの「/table」に貼り付け可能な形。

エクスポート対象を「直近N件」「○月分」で指定することも可能です。
