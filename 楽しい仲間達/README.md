# 楽しい仲間達（弊社全スタッフ）

このフォルダには、弊社の全スタッフの情報を置きます。

## 使い方

・**文字起こし・議事録での名前**：ここにいる人物を正とする。名前が出てきたときはこの一覧と照合する。
・**名前の誤り**：文字起こしで明らかにスタッフの話なのに一覧に該当する名前がない場合は、文字起こしの誤りとみなし、文脈から該当しそうな弊社スタッフ名に修正する（今後のMTGでも同様）。
・**個人の特性・配慮**：スタッフごとで気をつけることやアドバイス可能なことがあれば、ナレッジとして `ナレッジ/スタッフ_特性・アドバイス.md` などに保管する。

## このフォルダに置くもの

・スタッフ一覧（名前・役職・所属など）。ファイル名は任意（例：メンバー一覧.md、スタッフ一覧.md）。

**社員が増えた・減ったとき**は、ここを更新し、あわせて `参照情報/メンバー一覧.md` と `ナレッジ/スタッフ_特性・アドバイス.md` の要否を確認する。一覧は **`議事録_保守・更新ガイド.md`** に記載。
